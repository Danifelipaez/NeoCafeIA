# Granos Selectos Disponibles

## Orígenes y Características

### Etiopía Yirgacheffe
- **Perfil de Sabor**: Notas florales, cítricas y té negro. Acidez brillante, cuerpo ligero.
- **Recomendación**: Ideal para quienes prefieren cafés ligeros y afrutados. Perfecto para espresso o pour-over.

### Colombia Supremo
- **Perfil de Sabor**: Caramelo, nuez, chocolate. Acidez media, cuerpo completo.
- **Recomendación**: Excelente para lattes y cappuccinos. Apto para principiantes.

### Brasil Santos
- **Perfil de Sabor**: Dulce, bajo en acidez, notas de cacao y avellana.
- **Recomendación**: Gran opción para cafés con leche. Suave y accesible.

### Guatemala Antigua
- **Perfil de Sabor**: Ahumado, especiado, con notas de cacao oscuro.
- **Recomendación**: Para amantes del café intenso. Ideal para espresso doble.

### Costa Rica Tarrazú
- **Perfil de Sabor**: Cítricos, caramelo, cuerpo cremoso.
- **Recomendación**: Versátil para cualquier preparación. Alta calidad.

## Consejos de Preparación
- **Molienda**: Ajusta según el método (fina para espresso, media para drip).
- **Temperatura**: 90-95°C para extracción óptima.
- **Tiempo**: 25-30 segundos para espresso, 4-5 minutos para french press.
